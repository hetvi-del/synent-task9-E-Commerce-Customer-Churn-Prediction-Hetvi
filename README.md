# Task 9 - End-to-End Data Science Project

# E-Commerce Customer Churn Prediction & Retention Recommendation System

## Project Overview

This project is a complete end-to-end Data Science solution designed to predict customer churn in an e-commerce platform and provide retention recommendations.

The system helps businesses identify customers who are likely to leave and take proactive actions to improve customer retention.

## Business Problem

Customer churn directly impacts revenue and profitability.

The objective of this project is to predict churn risk and recommend strategies to retain valuable customers.

## Project Workflow

### 1. Data Collection

Collected E-Commerce Customer Churn Dataset.

### 2. Data Cleaning

- Missing Value Handling
- Duplicate Removal
- Data Type Validation

### 3. Exploratory Data Analysis

- Churn Analysis
- Customer Behavior Analysis
- Correlation Analysis
- Business Insights

### 4. Feature Engineering

Created advanced business features:

- Order Frequency
- Customer Value Score
- Loyalty Category
- High Distance Customer
- Engagement Score

### 5. Machine Learning

Implemented:

- Logistic Regression
- Decision Tree
- Random Forest

Selected the best-performing model.

### 6. Model Evaluation

Evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

### 7. Deployment

Developed an interactive Streamlit application.
